1. Для запуска необходимо установить следующие зависимости: 1. Dotenv, 2. Dancer2, 3. DateTime::Format::Strptime

2. В директории backend/bin/lib/ необходимо создать файл .env и заполнить его с помощью шаблона из файла backend/bin/lib/.env-example

3. Скрипт для запуска проекта лежит в директории: backend/bin/app.pl
